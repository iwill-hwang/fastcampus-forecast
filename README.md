## 🧑‍🎓 Forecast
네트워크 통신을 이용하여 날씨정보를 보여주는 앱.

## 내용
- 공공 데이터 OPEN API 사용해보기
- 테스트케이스를 이용하여 모델링 작업해보기
- 네트워크모듈 작성해보기
- 당겨서 새로고침 적용해보기
- 홈 스크린 위젯 적용해보기
